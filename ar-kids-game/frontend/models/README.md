# 3D Models

Place GLB/glTF model files here. Expected files:

- star.glb
- sun.glb
- cloud.glb
- rainbow.glb
- heart.glb
- rocket.glb
- teddy.glb
- butterfly.glb
- dinosaur.glb

Copy models to `public/models/` for Vite to serve them at `/models/`.

The MVP uses procedural Three.js meshes as fallback when GLB files are missing.
